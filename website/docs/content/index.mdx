# Bardo in one sentence

Bardo is an MCP toolbox that converts AI agents into capable, professional game masters for **any TTRPG system**.

## Why it works

Bardo keeps campaign truth in your local workspace instead of opaque chat context.

- World data lives in markdown files
- Game state lives in markdown files
- Entities, locations, quests, and history are persisted and auditable
- The GM process remains system agnostic because the model follows your files, not a locked schema
