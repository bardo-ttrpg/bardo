# Supported Agent Clients

Bardo is designed for **any MCP-capable client**.

## Known compatible examples

- VS Code + GitHub Copilot
- Claude Code
- Cursor
- Cline
- OpenCode
- Codex CLI
- Gemini CLI

If your agent supports MCP transport and tool-calling, Bardo can be integrated.
