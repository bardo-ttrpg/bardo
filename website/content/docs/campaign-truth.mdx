# Campaign truth

Bardo is opinionated about where authority lives so the product stays simple and predictable.

## Local source of truth

Your workspace remains the canonical source for campaign files and any local file operations.

## Remote service boundary

The hosted side governs bridge approvals, subscription state, rate limits, and remote tool execution.

## What this means in practice

If a question depends on your files, the answer should come from the local workspace. If it depends on access policy or metering, it comes from the remote service.
