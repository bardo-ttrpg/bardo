# Connect a client

The supported path is simple: install the bridge locally, choose your workspace, then approve the browser session from the dashboard.

## Expected flow

Install the bridge locally and open the dashboard in the browser.

Start a bridge session from your client. Bardo generates a user code and a verification URL.

Approve the session in the browser, then return to your client and use Bardo tools against the local workspace.

## Boundary

Your files stay local.

The hosted side handles subscription checks, metering, and the remote MCP service layer.
