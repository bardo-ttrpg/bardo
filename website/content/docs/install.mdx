# Install

Use the published install scripts, then confirm the CLI can authenticate and start bridge sessions.

## Shell

Run `curl -fsSL https://bardo.gg/install | sh` on macOS or Linux.

The installer clones the repo, installs dependencies with Bun, and writes local command shims for `bardo` and `bardo-mcp`.

## Windows

Run `irm https://bardo.gg/install.ps1 | iex` in PowerShell.

The PowerShell installer performs the same repo and shim setup for the Windows workflow.

## After install

Next, connect your client and approve a browser session from the dashboard.
