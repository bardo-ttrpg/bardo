# Credits and billing

Billing state now lives next to the dashboard instead of on a separate marketing page.

## Usage

The dashboard reports plan status, credits remaining, total MCP calls, and the next reset time.

## Subscription actions

If billing is configured, the dashboard exposes Clerk-powered subscribe and subscription management actions.

## Failure mode

If Clerk billing is unavailable, the dashboard still loads but billing actions stay disabled with an explicit message.
