# Marketplace Status

Bardo MCP is prepared for public registry and marketplace distribution as a local stdio MCP server.

## Public Identity

- Official MCP Registry name: `io.github.armando-andre/bardo`
- Public npm package: `@bardo/mcp`
- Website: https://bardo.gg

## Current Positioning

- Free to download.
- Requires active Bardo Pro access or the 3-day trial to use.
- Local stdio MCP server for this release.
- No hosted gameplay MCP endpoint in this release.

## Remote MCP Note

Do not list Bardo as a remote MCP server until it supports standards-compliant Streamable HTTP authorization for remote MCP use.

## Companion Skill

The Bardo MCP Skill is a free onboarding companion for agents. It teaches safe Bardo usage and does not bypass paid access.
