Use this guide for [Gemini CLI](https://geminicli.com/).

## Install Gemini CLI

Follow the latest official Gemini CLI install steps, then open the workspace you want Bardo to manage.

## Connect Bardo

From the workspace root:

```bash
bardo init --rulebook ./RULEBOOK.md
bardo validate
bardo connect --client gemini
```

Expected result: Bardo writes `.gemini/settings.json` for the current workspace.

## What Bardo writes

Bardo writes the local Gemini workspace config for you. If Gemini asks you to trust the workspace, approve it so the local config can load.

## Use Bardo well in Gemini

Troubleshooting:
- Restart Gemini CLI after connecting if Bardo is not listed yet.
- If Gemini says the workspace needs input, add the missing campaign material before continuing.
